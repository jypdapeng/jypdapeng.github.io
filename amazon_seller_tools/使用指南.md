# 🎯 亚马逊卖家智能辅助系统 - 中文使用指南

## 📖 系统介绍

这是一个专为**亚马逊中国卖家**打造的综合运营辅助工具，涵盖从选品到优化的全流程。

### ✨ 7大核心功能

1. **📊 选品分析** - 找到最有利可图的产品
2. **📈 运营管理** - 全面掌控店铺运营
3. **📦 库存管理** - 避免断货和积压
4. **💰 广告优化** - 提升广告ROI
5. **🔍 排名分析** - 追踪关键词排名
6. **🎯 竞品分析** - 了解竞争对手
7. **✍️ Listing优化** - 提升转化率

## 💡 使用教程

### 教程1：计算产品利润

**场景：** 评估一个产品是否值得做

1. 点击顶部 **"选品分析"**
2. 填写产品信息：
   - 售价：$29.99
   - 成本：$10
   - 佣金：15%
   - FBA费用：$4.50
3. 点击 **"计算利润"**
4. 查看结果：
   - ✅ 净利润：$12.75
   - ✅ 利润率：42%
   - ✅ ROI：127%

**结论：** 这个产品利润率很好，值得做！

---

### 教程2：评估产品机会

**场景：** 全面评估一个产品的市场机会

继续在选品分析页面：

1. 向下滚动到 **"产品综合评分"**
2. 填写产品数据（同上）
3. 填写市场数据：
   - 月销量：5000
   - 平均价格：$28
   - 平均评分：4.2
   - 评论数：500
   - 主要竞品：10个
4. 点击 **"分析产品"**
5. 获得综合评分和建议

---

### 教程3：使用Python直接调用

如果您更习惯用代码：

```python
# 打开Python
python

# 导入模块
from modules import ProductResearch
from config import Config

# 初始化
pr = ProductResearch(Config)

# 计算利润
result = pr.calculate_profit({
    'selling_price': 29.99,
    'cost': 10.0,
    'amazon_fee_percentage': 15,
    'fba_fee': 4.5
})

# 查看结果
print(f"利润: ${result['profit']}")
print(f"利润率: {result['profit_margin']}%")
```

### 案例2：广告ACoS从40%降到25%

**问题：** 广告花费太高，不赚钱

#### 第1步：诊断问题

使用 **广告优化** 模块分析：
- ACoS：40% ❌
- CTR：1.0% ✅
- CVR：5% ⚠️

**发现：** 点击率好但转化率低

#### 第2步：关键词分析

分析所有关键词：
- 高效词（ACoS<25%）：5个
- 中效词（ACoS 25-35%）：8个
- 低效词（ACoS>40%）：12个

#### 第3步：优化行动

1. 暂停12个低效关键词
2. 提高5个高效词出价
3. 优化产品页面提高CVR

#### 结果：

- 7天后ACoS降至32%
- 14天后ACoS降至28%
- 30天后稳定在25%

## ❓ 常见问题

### Q1：数据从哪里来？

**A：** 主要数据源：
- 亚马逊卖家中心
- 业务报告导出
- Jungle Scout / Helium 10
- 手动市场调研

### Q2：需要API吗？

**A：** 不需要！可以手动输入数据。如果想自动化，可以配置亚马逊SP-API（可选）。

### Q3：支持哪些站点？

**A：** 分析逻辑适用所有站点（美国、欧洲、日本等），但需要根据站点调整费用标准。

### Q4：数据安全吗？

**A：** 完全安全！所有数据存储在您本地，不上传到任何服务器。

### Q5：可以导出报告吗？

**A：** 可以！支持Excel导出（部分模块）。
